Postcard Print Page

Print / Save as PDF Instructions

  1. Press Ctrl+P (or Cmd+P on Mac)
  2. Set destination to "Save as PDF"
  3. Set paper size to "Letter"
  4. Set margins to "None"
  5. Enable "Background graphics"
  6. Disable headers and footers
  7. Save as PDF
